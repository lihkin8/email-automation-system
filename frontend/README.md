# Email Dashboard Frontend

React-based dashboard for the Email Automation System.

## Features

- Email tracking visualization
- Campaign management
- Recruiter management

## Prerequisites

- Node.js 16+
- npm or yarn

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/email-dashboard-frontend.git
   cd email-dashboard-frontend
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

## Development

```bash
npm start
# or
yarn start
```

## Building for Production

```bash
npm run build
# or
yarn build
```
